Decode your agents

Kibitz watches active Claude Code and Codex sessions, translates agent activity into clear live commentary, and lets you route prompts to any active session from one place.

Supported: macOS + Windows | Linux best effort
Live feed: Real-time summaries across multiple agent sessions.
Dispatch: Send prompts to existing sessions or start a new one instantly.
Workflow: Use Kibitz inside VS Code or from the terminal CLI.
